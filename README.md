# Sona Siju — Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Deploy on Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"New Project"** → Import your repository
4. Vercel auto-detects it as a static site — just click **Deploy**
5. Your site will be live at `https://your-repo-name.vercel.app`

## 📁 Project Structure

```
sona-portfolio/
├── index.html      # Main portfolio page (all-in-one)
├── vercel.json     # Vercel deployment config
└── README.md       # This file
```

## 🛠 Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (IntersectionObserver, animated counters)
- Google Fonts (Fira Code, Inter)
